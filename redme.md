#### Crear entorno virtual
```python
python -m venv .env
```

#### Activamos el entorno virtual
```shell
.\env\Scripts\activate
source .env/bin/activate
```

#### Instalar Flask
```python
pip install Flask
```

#### Mostrar paquetes instalados
```python
pip freeze
pip freeze > paquetes.txt
```
#### En caso de crear el proyecto 
``` 
pip install .r paquete.txt
```
